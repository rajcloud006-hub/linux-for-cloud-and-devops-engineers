# Linux for Cloud and DevOps

Practical Linux learning notes using a single building analogy to understand Linux, Cloud, DevOps and SRE concepts.

Instead of memorising commands, this guide focuses on understanding how a Linux system operates and how engineers troubleshoot real-world problems.

## The Building Analogy

Linux Server = Building

Process = Worker

Service = Department

Port = Door

Socket = Receptionist

Firewall = Security Guard

Logs = Security Cameras

Container = Temporary Office Room

Image = Architect's Blueprint
