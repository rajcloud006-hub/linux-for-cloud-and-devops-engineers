# Departments (Services)

A service is a department inside the building.

Examples:

SSH Department

Docker Department

Cron Department

A department provides a specific service to the building.

## Check Department Status

systemctl status ssh

## Start Department

sudo systemctl start ssh

## Stop Department

sudo systemctl stop ssh

## Restart Department

sudo systemctl restart ssh

## Building Analogy

Department = Service

Workers perform tasks for departments.

Example:

SSH Department

↓

SSH Workers

## Key Takeaway

Services provide functionality.

systemctl manages departments.
