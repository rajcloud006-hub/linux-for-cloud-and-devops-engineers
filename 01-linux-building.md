# Linux as a Building

The easiest way to understand Linux is to imagine a large office building.

Everything inside Linux has a purpose, just like every room in a building.

## Root (/)

The main entrance.

Everything starts here.

Example:

/

## /home

Private offices for users.

Example:

/home/ubuntu

Each user gets their own workspace.

## /etc

Building Control Room.

Contains instructions and configuration files that tell the building how to operate.

Examples:

SSH configuration

Network configuration

System configuration

## /var

Storage room for changing information.

Contains:

- logs
- caches
- temporary application data

## /var/log

Security camera recordings.

Stores events that happen throughout the building.

## /bin

Shared toolbox.

Contains essential tools used by everyone.

Examples:

ls

cp

mv

cat

## Key Takeaway

Every location in Linux exists for a reason.

Understanding purpose is more important than memorising paths.
